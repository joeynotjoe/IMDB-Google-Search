# IMDB-Google-Search
An Alfred workflow to search IMDB using a Google Search Suggestion backbone.

Based off of [Google Alfred 3 Workflow](http://www.packal.org/workflow/google-alfred3-workflow)

Usage and Modifiers
↩	|	open URL in browser
⌘	|	copy URL to clipboard
⌥	|	view Parental Guide
⇧	|	extract ID
⌃	|	full credits
fn	|	keywords
